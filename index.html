window.Telegram.WebApp. Функции

isVersionAtLeast(version)
Возвращает true, если приложение пользователя поддерживает версию Bot API, равную или выше указанной.
setHeaderColor(color)
Устанавливает цвет заголовка приложения в формате #RRGGBB.
setBackgroundColor(color)
Устанавливает цвет фона приложения в формате #RRGGBB.
setBottomBarColor(color)
Устанавливает цвет нижней панели приложения в формате #RRGGBB.
enableClosingConfirmation()
Включает диалог подтверждения при попытке закрыть мини-приложение.
disableClosingConfirmation()
Отключает диалог подтверждения при попытке закрыть мини-приложение.
enableVerticalSwipes()
Включает вертикальные свайпы для закрытия или минимизации мини-приложения.
disableVerticalSwipes()
Отключает вертикальные свайпы для закрытия или минимизации мини-приложения.
requestFullscreen()
Запрашивает открытие мини-приложения в полноэкранном режиме.
exitFullscreen()
Запрашивает выход из полноэкранного режима.
lockOrientation()
Блокирует ориентацию мини-приложения в текущем режиме (портрет или ландшафт).
unlockOrientation()
Разблокирует ориентацию мини-приложения, позволяя следовать за ориентацией устройства.
addToHomeScreen()
Предлагает пользователю добавить мини-приложение на главный экран.
checkHomeScreenStatus([callback])
Проверяет, поддерживается ли добавление на главный экран и добавлено ли мини-приложение.
onEvent(eventType, eventHandler)
Устанавливает обработчик событий для мини-приложения.
offEvent(eventType, eventHandler)
Удаляет ранее установленный обработчик событий.
sendData(data)
Отправляет данные боту и закрывает мини-приложение.
switchInlineQuery(query[, choose_chat_types])
Вставляет имя пользователя бота и указанный инлайн-запрос в поле ввода текущего чата.
openLink(url[, options])
Открывает ссылку во внешнем браузере.
openTelegramLink(url)
Открывает телеграм-ссылку внутри приложения Telegram.
openInvoice(url[, callback])
Открывает счет с использованием указанной ссылки.
shareToStory(media_url[, params])
Открывает редактор историй с указанным медиа.
shareMessage(msg_id[, callback])
Открывает диалог для отправки сообщения, предоставленного ботом.
setEmojiStatus(custom_emoji_id[, params, callback])
Открывает диалог для установки указанного эмодзи в качестве статуса.
requestEmojiStatusAccess([callback])
Запрашивает разрешение на управление статусом эмодзи пользователя.
downloadFile(params[, callback])
Отображает всплывающее окно для загрузки файла.
showPopup(params[, callback])
Показывает всплывающее окно, описанное параметрами.
showAlert(message[, callback])
Показывает сообщение в простом алерте с кнопкой "Закрыть".
showConfirm(message[, callback])
Показывает сообщение в окне подтверждения с кнопками "ОК" и "Отмена".
showScanQrPopup(params[, callback])
Показывает всплывающее окно для сканирования QR-кода.
closeScanQrPopup()
Закрывает всплывающее окно для сканирования QR-кода.
readTextFromClipboard([callback])
Запрашивает текст из буфера обмена.
requestWriteAccess([callback])
Запрашивает разрешение на отправку сообщений пользователю.
requestContact([callback])
Запрашивает номер телефона пользователя.
ready()
Уведомляет Telegram, что мини-приложение готово к отображению.
expand()
Разворачивает мини-приложение на максимальную доступную высоту.
close()
Закрывает мини-приложение.
